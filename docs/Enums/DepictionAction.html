DepictionAction

public enum DepictionAction

Enum of possible actions for a depiction action

  • openURL(url:external:)

    Handler for the open URL. An example is https://twitter.com/elihwyma

    Author

    Amy

    Version

    1.0

    Important

    This will only register for the URL schemes HTTP and HTTPS

    Declaration

    Swift

    case openURL(url: URL, external: Bool)

    Parameters

    url

    The URL to open
    external

    Whether the link should be opened in a built in web view or an external open should be opened
  • openDepiction(url:)

    Handler for the open depiction option. An example is nd://open-depiction=https://chariz.com/buy/jellyfish/depiction.json

    Author

    Amy

    Version

    1.0

    The recommended implementation is opening this in a second view controller with just a depiction view in it

    Declaration

    Swift

    case openDepiction(url: URL)

    Parameters

    url

    The package bundle ID to open
  • openPackage(bundle:)

    Handler for the open package option. An example is nd://open-package=com.amywhile.aemulo

    Author

    Amy

    Version

    1.0

    Declaration

    Swift

    case openPackage(bundle: String)

    Parameters

    bundle

    The package bundle ID to open
  • addRepo(url:)

    Handler for the add repo action. An example is nd://add-repo=https://repo.chariz.com

    Author

    Amy

    Version

    1.0

    Declaration

    Swift

    case addRepo(url: URL)

    Parameters

    url

    The repo URL that is to be added
  • custom(action:)

    This is to be used for any custom action you would like to invoke in your depictions if you would like to add your own custom views. An example action is zebra://open-installed-contents

    Author

    Amy

    Version

    1.0

    Important

    This may also be called for invalid actions in a depiction, you must handle invalid actions on your side

    Declaration

    Swift

    case custom(action: URL)

    Parameters

    action

    The custom defined action
  • actionError(error:action:)

    Error handler for depiction actions.

    Author

    Amy

    Version

    1.0

    This error will be returned if there is an error when parsing an action, such as an invalid depiction URL for open-depiction. It is suggested to show these errors in an alert.

    Declaration

    Swift

    case actionError(error: String, action: String)

    Parameters

    error

    A string describing the error
    action

    The action that resulted in the error
