  • Attachment

    A dynamic attachment for different themes and device sizes.

    DepictionKit supports customising the media that will show in the Depiction based on if its an iPhone or iPad and if the Depiction is in light or dark mode.

    Important

    If the ipad dict is not present it will fall back to iphone

    Author

    Amy

    Version

    1.0

    Declaration

    Swift

    public final class Attachment

    Parameters

    iphone

    [String: String]; Required keys are lightand dark
    ipad

    [String: String]?; Required keys are light and dark

  • FontWeight

    The weight for fonts to use in text based views. This matches the system font.

    Supports values are:

    • ultralight
    • thin
    • light
    • regular
    • medium
    • semibold
    • bold
    • heavy

    • black
