DepictionContainer

final public class DepictionContainer : UIView

Depiction Container is the main view you need to implement to show depictions in your app

It supports multiple methods of providing a depiction to display, support for light and dark mode, custom tint colours and multiple delegate methods for interacting with the depiction.
